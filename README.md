
# Job Portal UI

A modern, responsive job search and filtering interface built with **React 18**, **TypeScript**, and **Tailwind CSS**.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

**Open browser:** `http://localhost:5173`

## ✨ Features

- 🔍 **Job Search & Filtering** - Search by title, filter by category, location, job type & salary
- 📱 **Fully Responsive** - Desktop sidebar filters + mobile filter drawer
- ⚡ **Real-time Updates** - Instant filtering as you adjust parameters
- 🎨 **Modern UI** - Clean, professional design with Radix UI components
- 🌙 **Theme Support** - Built-in CSS variable system for easy customization
